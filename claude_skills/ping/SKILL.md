# Ping AI Provider

Test connectivity with specified AI provider.

## Usage

The first argument must be the provider name:
- `codex` - Test Codex
- `claude` - Test Claude

## Execution (MANDATORY)

```
Bash(ping $PROVIDER)
```

## Examples

- `/ping codex`
- `/ping claude`
