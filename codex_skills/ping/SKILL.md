# Ping AI Provider

Test connectivity with specified AI provider.

## Usage

The first argument must be the provider name:
- `codex`
- `claude`

## Execution (MANDATORY)

```bash
ping $PROVIDER
```

## Examples

- `/ping codex`
- `/ping claude`
